# Course Coordination Web Portal

## Overview
A full-stack web application developed as a graduation project to facilitate course coordination, academic collaboration, event management, and resource sharing.

## Features
- User authentication and authorization
- Course coordination
- Event management
- Discussion forums
- Resource sharing
- User profile management

## Technologies
- PHP
- MySQL
- HTML
- CSS
- JavaScript
